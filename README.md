# CodeVault - Code Snippet Manager

**Framework**: Jetpack Compose + Room
**Architecture**: MVVM + Repository

## Dependencies

- Room database for local storage
- Dagger Hilt for dependency injection
- Firebase Firestore for initial code snippets
- Jetpack Compose Navigation

## Components & Layout

- [ ]  Snippet list with categories/tags filters
- [ ]  Snippet detail screen with syntax-highlighted code display
- [ ]  Add/Edit snippet form with title, language, code, tags

## Functionalities

- [ ]  Store code snippets locally with title, content, and tags.
- [ ]  Support categorized tagging for easy search and filtering.
- [ ]  Edit, delete, and share snippets.
- [ ]  Implement syntax highlighting using code editor library or custom styling.
